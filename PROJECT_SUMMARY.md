## 🎮 What's Included in the App

### Team Builder Tab
- Add up to 8 champions
- Filter by trait or search by name
- Auto-calculate trait synergies
- Calculate total unit cost
- Save teams to device
- View all saved teams
- Delete unwanted teams

### Meta Tier List Tab
- Browse S/A/B tier compositions
- See win rates and pick rates
- Filter by set
- Read strategy descriptions
- Current meta for Set 16

### Set Browser Tab
- View all 4 sets (14, 15, 16, 17)
- Browse 50+ champions
- See 30+ traits with descriptions
- View champion costs and traits
- Set status (current, legacy, upcoming)

### Data Storage
- Teams saved in browser localStorage
- No data sent to servers
- Device-specific storage
- Survives browser restart
- ~500+ teams can be stored

### Offline First
- Works with zero internet
- All data bundled in app
- Instant load times
- No API calls
- Perfect for tournaments, LAN, travel
